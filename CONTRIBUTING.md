# Contributing to OSINT-BIBLE
Thanks for your interest in improving this guide!

## How to Contribute

Adding a Tool or Resource
Fork the repository
Add the tool to the appropriate section in README.md
Include: Tool name, description, link, and category
Verify the link works
Submit a pull request

## Reporting Broken Links

Open an issue with the title "Broken link: [tool name]" and include the broken URL and section.

## Suggesting New Categories
Open an issue with the proposed category name and at least 3-5 resources that would fit.

## Guidelines

Verify all links before adding
Keep descriptions concise and factual
Respect privacy and ethical boundaries in OSINT research
Only include publicly available, legal resources
